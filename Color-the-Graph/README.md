## Guide run program Color the Graph ##

### On Linux ###
`
Turn on terminal and follow the steps below
`

* Step 1:
`
sudo apt install g++
`

* Step 2:
`
cd Color-the-Graph
`

* Step 3:
`
g++ main.cpp (or sudo g++ main.cpp)
`

* Step 4:
`
./a.out
`

### On Windows ###
`
Download source and open with IDE (C-Free, CodeBlock, DevC++, ...) to run.
`
